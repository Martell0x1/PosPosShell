# PosPosShell Project

## Features Implemented

- Prompt display
- Command execution using fork + exec
- Built-in commands:
  - cd
  - pwd
  - exit
  - history
- Background execution (&)
- Signal handling (Ctrl+C)

## Compilation

make

## Clean
make clean

## run Debug
make debug

## Run

./myShell
